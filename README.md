## 🚀 参与方式

### 方法一：网页直接编辑

**操作步骤（简洁流程）**

1. **打开Repository**：[👉 点击访问](https://github.com/yuchenwu73/The-15th-Five-Year-Plan-Study)
2. **选择文件并编辑**：
   - 进入 `政策文件/`目录
   - 点击任意文件（如 `01-发展新质生产力.md`）
   - 点击右上角Edit按钮
   - 滚动到**底部**添加你的解读
3. **Commit & PR**：
   - 填写Commit message（如：`添加对01-发展新质生产力的理解-张三`）
   - 选择Create a new branch
   - 点击Propose changes → Create pull request
   - 提交完成

### 方法二：本地Git操作（适合熟悉Git的同学）

```bash
# 克隆repository
git clone https://github.com/yuchenwu73/The-15th-Five-Year-Plan-Study.git
cd The-15th-Five-Year-Plan-Study

# 创建branch
git checkout -b 你的名字-政策理解

# 编辑政策文件，底部添加解读

# 提交
git add 政策文件/你修改的文件.md
git commit -m "添加对01-发展新质生产力的理解-张三"
git push origin 你的名字-政策理解

# 在GitHub上创建Pull Request
```

详细操作步骤请查看 [参与指南.md](./参与指南.md)

## 💡 解读撰写建议

1. **准确理解**：用自己的语言准确表述政策内涵，避免简单照搬原文
2. **专业关联**：从计算机、网络安全等专业视角进行深度分析
3. **结合实践**：联系个人学习、科研与未来职业发展进行思考
4. **注重质量**：避免空泛表述，提出具体可行的实践方案

**解读模板结构：**

- 我的理解：对政策核心内涵的个人解读
- 专业关联与研究方向：结合专业背景与研究方向分析政策关联性
- 我能做什么：提出个人层面的实践思路
